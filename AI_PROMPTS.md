## AI Prompts Used

AI tools were used as a development assistant to support planning, structure, and documentation. The following prompts guided the process:

- "Design a scalable folder structure for a React (Vite) application with feature-based organization, including pages for a video listing, video detail, and creator profile."

- "Suggest a clean architecture for a mini YouTube-style frontend using React, focusing on reusable components, separation of concerns, and static mock data."

- "Provide a lightweight implementation approach for routing, component breakdown, and data modeling for a simplified video streaming UI."

- "Draft a concise README for a Vite + React + TypeScript project including setup instructions, features, and technology stack explanation."

All AI-generated suggestions were reviewed, adjusted, and implemented manually. Final code structure, UI decisions, and component composition reflect personal understanding and refinement rather than direct output.
